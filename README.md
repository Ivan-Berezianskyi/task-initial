# IT Salary Prediction App

Цей проєкт — це інтерактивний вебдодаток на базі **Streamlit**, який передбачає заробітну плату ІТ-спеціаліста на основі введених характеристик (досвід, мова програмування, рівень англійської, посада тощо) за допомогою моделі лінійної регресії.

## Технологічний стек

* **Python**
* **Streamlit**
* **Pandas & NumPy**
* **Matplotlib**
* **Joblib**
* **python-dotenv**

---

## Як запустити локально

### 1. Створення віртуального середовища

Створи віртуальне середовище за допомогою команди:
```bash
python -m venv venv
```

**Активація для Windows:**
```bash
venv\Scripts\activate
```

**Активація для macOS/Linux:**
```bash
source venv/bin/activate
```

### 2. Встановлення залежностей

Встанови необхідні бібліотеки через pip:
```bash
pip install -r requirements.txt
```

### 3. Налаштування середовища

Створи файл `.env` у кореневій папці проєкту та вкажи шлях до твоєї натренованої моделі. Наприклад:
```env
MODEL_PATH=models/linear_regression_model.joblib
```

### 4. Запуск додатку

У терміналі виконай команду:
```bash
streamlit run app.py
```

Додаток автоматично відкриється у твоєму браузері за адресою: `http://localhost:8501`.
